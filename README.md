# Telecomm IaC Provisioning Project

## Description
This project automates the provisioning and deployment of a Telecomm system using Infrastructure as Code (IaC).

## Tools Used
- Terraform
- AWS EC2
- Docker
- Docker Hub
- Kubernetes (Minikube)
- GitHub Actions

## Workflow
1. Code pushed to GitHub
2. Infrastructure provisioned using Terraform
3. Application containerized using Docker
4. Image pushed to Docker Hub via CI/CD
5. Deployed on Kubernetes using Minikube

## Outcome
Fully automated, repeatable, and scalable deployment pipeline.
